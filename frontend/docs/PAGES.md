# 프론트엔드 페이지 구조 및 기능 설명

## 개요
이 문서는 Dash 프로젝트의 프론트엔드 페이지 구성과 각 페이지의 주요 기능을 설명합니다.

## 페이지 구조

### 1. 대시보드 (Dashboard)
- **경로**: `/dashboard`
- **파일**: `src/views/Dashboard.vue`
- **기능**:
    - **사용자 환영**: 로그인한 사용자의 이름을 표시합니다.
    - **통계 요약**: 사용자의 알고리즘 풀이 수, 연속 학습일(Streak), 현재 티어를 카드 형태로 보여줍니다.
    - **빠른 이동**: 게시판 및 프로필 페이지로 이동하는 바로가기를 제공합니다.
    - **알림**: 알고리즘 기록 연동 등 중요한 알림이나 팁을 표시합니다.

### 2. 게시판 (Board)
커뮤니티 기능을 담당하며 다음 3가지 뷰로 구성됩니다.

#### 2.1 게시글 목록 (List)
- **경로**: `/boards`
- **파일**: `src/views/BoardList.vue`
- **기능**:
    - 전체 게시글을 번호, 제목, 작성자, 작성일 순으로 목록화하여 보여줍니다 (API: `findAll`).
    - '글쓰기' 버튼을 통해 새 글 작성 페이지로 이동할 수 있습니다.
    - 게시글 클릭 시 상세 페이지로 이동합니다.

#### 2.2 게시글 상세 (Detail)
- **경로**: `/boards/:id`
- **파일**: `src/views/BoardDetail.vue`
- **기능**:
    - 선택한 게시글의 제목, 작성자, 작성일, 조회수, 본문을 표시합니다 (API: `findById`).
    - **작성자 전용 기능**: 본인이 작성한 글인 경우 '수정', '삭제' 버튼이 노출됩니다.

#### 2.3 게시글 작성/수정 (Form)
- **경로**: `/boards/write` (작성), `/boards/edit/:id` (수정)
- **파일**: `src/views/BoardForm.vue`
- **기능**:
    - 제목과 내용을 입력받아 글을 작성하거나 수정합니다 (API: `create`, `update`).
    - URL 파라미터 유무에 따라 작성 모드와 수정 모드를 자동으로 구분합니다.

### 3. 프로필 (Profile)
- **경로**: `/profile`
- **파일**: `src/views/Profile.vue`
- **기능**:
    - **정보 조회**: 사용자의 이름, 이메일 등 기본 정보를 표시합니다.
    - **정보 수정**: 사용자 이름을 수정할 수 있습니다 (API: `update`).
    - **회원 탈퇴**: 계정을 영구적으로 삭제하는 기능을 제공합니다 (API: `delete`).

## 디자인 스타일
- **Tailwind CSS** 기반의 다크 모드를 기본으로 합니다 (`bg-slate-950`, `text-slate-100`).
- **일관된 헤더**: 모든 페이지에 `CommonHeader` 컴포넌트를 사용하여 통일성을 부여했습니다.
- **애니메이션**: 페이지 진입 시 부드러운 페이드 인/업 애니메이션을 적용하여 고급스러운 사용자 경험을 제공합니다.
